### Analytical Questions

## Uploading Your Own Documents

1. **Supported Formats**: PDF files
2. **Recommended Size**: Under 50MB per file
3. **Content Type**: Text-based PDFs work best
4. **Language**: English content produces best results

## Tips for Better Results

### Document Quality
- Use clear, well-formatted PDFs
- Avoid image-heavy documents (unless OCR processed)
- Ensure text is selectable in the PDF

### Question Formulation
- Be specific and clear
- Ask one question at a time
- Use natural language
- Reference document context when needed

### Parameter Tuning
- **Lower Temperature (0.0-0.3)**: More factual, focused answers
- **Higher Temperature (0.7-1.0)**: More creative, diverse responses
- **Chunk Size**: 500-1000 characters work well for most documents

## Troubleshooting

### Common Issues
1. **No relevant information found**: Try rephrasing your question
2. **Generic responses**: Check if documents were loaded properly
3. **Slow responses**: Consider reducing document size or chunk count

### Getting Better Results
1. Load multiple related documents for comprehensive coverage
2. Use specific terminology from your domain
3. Ask follow-up questions to drill down into topics
4. Experiment with different temperature settings
